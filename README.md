# The Technological Blueprint of Canada: 400 Years of Canadian Technology

An interactive, premium digital exhibition exploring Canada's history through five key technological artifacts spanning four centuries.

## Project Structure

- `introductory_label.md`: The exhibition's curatorial statement and design framework.
- `presentation/`: Core presentation files.
  - `index.html`: The interactive slideshow presentation structure and logic.
  - `styles.css`: Visual styling, layout, typography, and custom transitions.
  - `images/`: High-quality historical and modern graphics representing the five artifacts.
  - `build_ppt.py`: Python script used to output the slides as a PowerPoint presentation.
  - `The_Technological_Blueprint_of_Canada.pptx`: Generated PowerPoint presentation.

## Core Features

1. **Interactive Timeline:** Fast jump navigation to specific eras via the Overview and Summary slides.
2. **Curatorial Intro & Restart Slide:** Seamless museum-style introduction and a reset screen at the end to restart or review citations.
3. **Animated Fun Facts:** Splitting-image animation overlays revealing interesting historical context on gallery slides.
4. **Grandfather Clock Transitions:** Era-specific time-travel spin animations when crossing historical boundaries.
5. **Academic Citations:** MLA 9-compliant Works Cited page with hanging indents (0.5 inches) and hyperlinked sources.
